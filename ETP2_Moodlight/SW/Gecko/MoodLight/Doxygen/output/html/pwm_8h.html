◆ PWM_init()

void PWM_init ( )

Initialize TIMER0, TIMER2, LETIMER0.

All the peripherals are initialized.

Note
TIMER0 is used to generate PWM signal with a frequency of 50kHz for solution 2.
TIMER2 is used to generate interrupts for PI controller with a frequency of 10kHz for solution 2.
LETIMER0 is used to generate PWM signal with a frequency of 400Hz for solution 0
